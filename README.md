# ED3-TP-FINAL
Trabajo práctico final de Electronica Digital 3 del grupo The rolling clocks - FCEFyN 2026 

# Integrantes:
- Acosta Facundo
- Caldera Pedro
- Sorbera Josué

# Descripción del proyecto:
El proyecto consiste en el movimiento de un servomotor en donde estaría conectada una cámara de vigilancia. 
El movimiento del servomotor se controla a través de un joystick, el cual se conecta a una placa LPC1769.
